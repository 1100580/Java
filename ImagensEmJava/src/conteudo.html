Imagens em Java
1. Mostra uma imagem num JLabel.
Necessário fazer o Build do Projecto para copiar os ficheiros de imagens de src para classes.
Para criar um jar correcto (em dist): Projecto > Properties > Run > Main Class. 		LabelComImagem.java
2. Mostra uma imagem num JLabel e troca a imagem usando setIcon().
 MostrarImagemJLabel.java
3. O tamanho do JLabel é maior que a imagem. O JLabel é colocado opaco e com cor de fundo. MostrarImagemJLabel2.java
4. Mostra uma imagem num JPanel. O tamanho da imagem acompanha as variações do tamanho do JPanel.
Troca a imagem invocando repaint(). 		MostrarImagemJPanel.java
5. Vários JLabel são colocados parcialmente sobrepostos num JLayeredPane.
Ver como usar Layered Panes. 		MostrarImagensJLayeredPane.java
6. Araste de JLabels num JLayeredPane. Os JLabels são posicionados com o método setBounds().
Durante o arraste os JLabels sobem para a camada Drag Layer. No fim voltam para a camada de partida.
Ver como usar Layered Panes. 		ArrasteImagemJLayeredPane.java
7. Araste de JLabels num JLayeredPane. Colocou-se a restrição de os Jlabels só poderem ser movidos para quadrados da mesma cor.
Se o movimento do JLabel é válido a sua posição final é ajustada no quadrado que contém o ponteiro do rato. 		ArrasteImagemJLayeredPane2.java
8. Se o movimento do JLabel é inválido o regresso à sua posição de partida é efectuado por um thread. ArrasteImagemJLayeredPane3.java
